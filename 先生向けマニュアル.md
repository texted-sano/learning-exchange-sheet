# 学習交流シート　先生向けマニュアル

このツールは、**生徒が授業のふりかえりを書き、先生がコメントや評価を返し、生徒があとから見返せる**「デジタル学習ノート」です。

データはすべて、先生ご自身のGoogleスプレッドシート（Googleの表計算ソフト）に保存されます。他の場所には保存されないので、安心してお使いいただけます。

## 1. 準備するもの

- **Googleアカウント**（Gmailをお使いなら、すでにお持ちです）
- **パソコン**（はじめの設定はパソコンで行ってください。日々の利用はタブレットでも大丈夫です）
- 配布された2つのファイル
  - `Code.gs` … スプレッドシートに組み込む「仕組み」の部分
  - `index.html` … 先生や生徒が画面で操作する部分

## 2. はじめの設定（最初に一度だけ）

ここが一番の山場です。**約15分**、手順どおりに進めれば必ずできますので、ひとつずつゆっくり進めてください。

### 手順1　新しいスプレッドシートを作る

1. インターネットで「**Googleスプレッドシート**」を開きます（ https://sheets.google.com ）
2. 「＋（空白のスプレッドシート）」を押して、新しい表を作ります
3. 左上の「無題のスプレッドシート」を押して、わかりやすい名前に変えます（例：「6年1組 学習交流シート」）

### 手順2　「仕組み」を組み込む

1. スプレッドシートの上のメニューから「**拡張機能**」→「**Apps Script**」を押します
   - 新しい画面（タブ）が開きます。プログラムを書くための画面ですが、書く必要はありません。**貼り付けるだけ**です。
2. 画面の中央に、すでに数行の文字が入っています。**すべて選んで消してください**（文字の上でクリックしてから、キーボードの Ctrl と A を同時に押す →Deleteキー）
3. 配布された `Code.gs` ファイルをメモ帳などで開き、**中身をすべてコピー**して、いま空にした場所に**貼り付け**ます
4. 画面上部の**フロッピーディスクの形をした保存ボタン**を押します

### 手順3　シートを初期化する

1. スプレッドシートの画面（表の画面）に戻り、**ページを再読み込み**します（キーボードのF5キー、またはブラウザの丸い矢印ボタン）
2. 少し待つと、上のメニューに「**学習交流シート**」という項目が現れます
   - 出てこないときは、10秒ほど待ってからもう一度再読み込みしてください
3. 「学習交流シート」→「**シートを初期化**」を押します
4. 初回は「承認が必要です」という確認画面が出ます。これは「このスプレッドシートを、この仕組みが読み書きしてもよいですか？」というGoogleからの確認です。
   - 「続行」→ ご自身のアカウントを選ぶ →（「このアプリはGoogleで確認されていません」と出た場合は「詳細」→「（プロジェクト名）に移動」）→「許可」の順に進めてください
5. もう一度「学習交流シート」→「シートを初期化」を押します。「初期化が完了しました」と出れば成功です
6. 画面の下を見ると、「**設定**」「**名簿**」「**授業**」「**テーマ**」「**回答**」という5つのタブ（シート）ができています

### 手順4　名簿に生徒を登録する

1. 下のタブから「**名簿**」を開きます
2. **B列（生徒名）に、生徒の名前を1行に1人ずつ**入力します。A列の出席番号は、入れても入れなくても構いません
3. **C列（パスワード）は空のままにしてください。** 生徒が初めて使うときに、自分で決めます

> 【個人情報が気になる場合】名前を入れず、**A列の出席番号だけ**で登録することもできます。その場合、画面には「5番」のように番号で表示されます。
> ※ 番号だけで使いはじめた生徒に、あとから名前を追記すると「別の生徒」として扱われ、それまでの記録と繋がらなくなります。名前で使うか番号で使うかは、**使いはじめる前に決めてください。**

### 手順5　インターネットから使えるように公開する

1. もう一度「拡張機能」→「Apps Script」の画面を開きます
2. 右上の青い「**デプロイ**」ボタン →「**新しいデプロイ**」を押します
3. 左上の**歯車マーク**を押して「**ウェブアプリ**」を選びます
4. 次のように設定します
   - 「次のユーザーとして実行」→ **自分**
   - 「アクセスできるユーザー」→ **全員**（生徒がGoogleにログインせずに使えるようにするためです）
5. 「デプロイ」を押します
6. 「**ウェブアプリのURL**」として、`https://script.google.com/macros/s/……/exec` という長い文字列が表示されます。**これがこのクラス専用の「接続先URL」です。「コピー」ボタンでコピーして、メモ帳などに保存しておいてください**

### 手順6　画面を開いて、先生のパスワードを決める

1. 配布された `index.html` を**ダブルクリック**して開きます（ブラウザで画面が開きます）
2. 「接続先URL」の欄に、手順5でコピーしたURLを**貼り付けて**、「接続する」を押します
3. はじめてなので「**初回ログイン**」という画面が出ます。**先生用のパスワード（4文字以上）**を決めて、2回入力してください
4. 先生の画面が開けば、設定はすべて完了です！

> パスワードは忘れないようにメモしてください。「設定」タブからいつでも変更できます。

## 3. 毎日の使い方（先生）

`index.html` を開く → 接続先URLを貼り付けて「接続する」→ パスワードでログイン、が毎回の入り方です。

画面の上に4つのタブがあります。

### 「授業を作成」タブ
- 日付・タイトル・**発問**（生徒に考えてほしい問いかけ）を入力して保存します
- **テーマ**を選ぶと、あとで生徒の成長をテーマごとに振り返れます（選ばなくても使えます）
- 「今すぐ生徒に公開する」にチェックを入れておくと、保存した瞬間から生徒が回答できます
- 下の「テーマ管理」で、テーマの追加・削除ができます（すでに授業で使っているテーマは削除できません）

### 「授業一覧」タブ
- 作った授業が並びます。日付の横には、その授業を**作成した時刻**も表示されます
- 「日付順」「テーマ順」で並べ替えられます。右上の「**お気に入りのみ表示**」にチェックを入れると、お気に入り登録した授業だけに絞り込めます
- 各授業の**テーマは、あとからいつでも変更できます**（作成時に選び忘れても、ここで選び直せます）
- 右上の**☆マーク**を押すと、その授業を「**お気に入り**」に登録できます（★に変わります）。よく使う授業に目印をつけておくのに便利です
- 「**回答を見る・評価する**」を押すと、クラス全員の提出状況が一目でわかります
- 提出された回答には、**評価（1〜10）・いいね・コメント**をつけて「保存」を押します。保存すると、その生徒のカードは小さくたたまれるので、まだ見ていない回答が探しやすくなっています
- 各授業の右下の「**削除**」を押すと、その授業を削除できます。削除した授業は**生徒の画面（「授業」「きろくを見る」の両方）から見えなくなります**。ただしデータは消えず、いつでも元に戻せます（下記）

#### 削除した授業を戻す・完全に消す
- 授業一覧の右上「**削除した授業**」を押すと、削除した授業の一覧が開きます
- 「**元に戻す**」を押すと、その授業が復活し、生徒の画面にも再び表示されます
- 「**完全に削除**」を押すと、スプレッドシートからも消えます。**この操作だけは元に戻せません**のでご注意ください（生徒が提出した回答そのものは「回答」シートに残ります）

### 「生徒管理」タブ
- **生徒用の接続コード**の発行（詳しくは次の章）
- 生徒の追加、名簿からの削除
- パスワードを忘れた生徒への対応（「7. 困ったときは」参照）
- 「**テーマ**」ボタン … その生徒の回答を、テーマごとにまとめて見られます
- 「**評価まとめ**」ボタン … テーマごとの評価の平均を、レーダーチャート（クモの巣のようなグラフ）で見られます

### 「設定」タブ
- 先生の名前、クラス名、パスワードをいつでも変更できます

## 4. 生徒に使ってもらう準備

生徒が使うために必要なものは2つです。

1. **`index.html` ファイル**（先生と同じもの）… 生徒の各端末に配布します（学校の共有フォルダに置く、USBで配る、など普段の方法で構いません）
2. **接続コード** … 「生徒管理」タブの一番上、「**生徒用の接続コード**」の「接続コードを発行」を押すと作られます。「コピーする」を押して、生徒に伝えてください（黒板に貼る、共有フォルダにテキストで置く、など）

> **接続コードとは？** このクラスのスプレッドシートに繋がるための「合言葉」です。このコードを知らない人は、たとえこのツールを持っていても、クラスのデータには一切触れません。
>
> **もしコードが外部に漏れてしまったら**、「**接続コードを再発行**」を押してください。新しいコードが作られ、**古いコードはその場で使えなくなります**（生徒には新しいコードを配り直してください）。

## 5. 生徒の画面での使い方

生徒には次のように説明してください。

1. `index.html` を開く
2. 「**生徒の方はこちら（接続コードで入る）**」を押す
3. 先生からもらった**接続コードを貼り付けて**「接続する」
4. 名簿から**自分の名前（または出席番号）を選ぶ**
5. **はじめてのときだけ**、自分のパスワード（4文字以上）を決めて登録します。2回目からは、そのパスワードでログインします
6. 「授業」タブで、先生の発問に自分の考えを書いて「送信する」
7. 「きろくを見る」タブで、自分の過去の回答と、先生からの評価・いいね・コメントが見られます。各授業は最初たたまれた状態で並び、「**開く**」を押すと、そのときの回答と先生からのコメントが表示されます

> 回答は何度でも書き直せます（送信し直すと上書きされます）。
> 先生が授業を削除すると、その授業の記録は「きろくを見る」からも見えなくなります（先生が元に戻せば、また表示されます）。

## 6. 便利な機能

- **テーマで振り返る** … 授業にテーマ（平和、命、多様性など）をつけておくと、「この子は『命』の授業でどんなことを考えてきたかな」という縦の振り返りが簡単にできます。テーマは授業を作ったあとでも、授業一覧からいつでも変更できます
- **お気に入り** … 授業一覧で☆マークを押すと、その授業に目印をつけられます。「お気に入りのみ表示」で、よく使う授業だけをすぐ呼び出せます
- **レーダーチャート** … テーマごとの評価の平均がグラフになります。学期末の所見づくりの参考になります
- **番号だけの匿名運用** … 名前を使わず出席番号だけで運用できます

## 7. 困ったときは

| 困りごと | 対処 |
|---|---|
| 生徒がパスワードを忘れた | 「生徒管理」でその生徒の「**パスワードをリセット**」を押します。次に生徒がログインするとき、新しいパスワードを自分で決め直せます。先生が決めて伝えたい場合は「**パスワードを指定する**」を使います |
| 先生がパスワードを忘れた | スプレッドシートの「**設定**」シートを開き、「教師パスワード」の値の欄を**空**にして保存します。次に接続すると、初回と同じパスワード設定画面が出ます |
| 「試行回数が多いため…」と出る | パスワードの間違いが続いたときの安全装置です。**5分ほど待ってから**やり直してください |
| 「学習交流シート」メニューが出ない | スプレッドシートのページを再読み込み（F5）して、少し待ってください |
| 「生徒用URLが無効です」と出る | 接続コードが古い可能性があります。先生が「接続コードを再発行」した後は、**新しいコード**を配り直してください |
| 「不明な action です」と出る | 「仕組み」（Code.gs）が新しくなったのに、公開のし直しがされていないときに出ます。次項の【重要】の手順を行ってください |
| 動きがゆっくりに感じる | Googleの仕組み上、ボタンを押してから1〜2秒かかるのは正常です。故障ではありません |

### 【重要】`Code.gs` を新しいものに差し替えたとき

配布元から新しい `Code.gs` を受け取って貼り替えたときは、**保存するだけでは反映されません。** 必ず次を行ってください。

1. Apps Scriptの画面で「**デプロイ**」→「**デプロイを管理**」
2. **鉛筆マーク**（編集）を押す
3. 「バージョン」を「**新しいバージョン**」に変える
4. 「**デプロイ**」を押す

> このとき**接続先URLは変わりません**。生徒への再配布は不要です。

### データについて

- すべてのデータはスプレッドシートの中にあります。**スプレッドシートを消さない限り、データが消えることはありません**
- 心配なときは、スプレッドシートの「ファイル」→「コピーを作成」でいつでも控えが取れます
- 生徒を名簿から削除しても、その生徒の過去の回答は「回答」シートに残っています

## 最後に（大切なお願い）

- このツールは**試作品**です。生徒のパスワードは、スプレッドシートの中にそのままの文字で保存されています。**銀行やスマホと同じパスワードを使わないよう**、生徒に伝えてください
- スプレッドシートの**共有設定は「自分のみ」のまま**にしてください。他の人に共有すると、名簿やパスワードが見えてしまいます
